vindula.comissao Installation
-----------------------------

To install vindula.comissao using zc.buildout and the plone.recipe.zope2instance
recipe to manage your project, you can do this:

* Add ``vindula.comissao`` to the list of eggs to install, e.g.:

    [buildout]
    ...
    eggs =
        ...
        vindula.comissao
       
* Re-run buildout, e.g. with:

    $ ./bin/buildout
